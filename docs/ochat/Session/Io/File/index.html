Module Io.File

val read : Bin_prot_utils_eio.path -> t

read path loads a snapshot produced by write. The value is automatically migrated to the latest schema.

val write : Bin_prot_utils_eio.path -> t -> unit

write path session serialises session to path using `bin_dump` with a header. The file is created (0600) or truncated atomically.
