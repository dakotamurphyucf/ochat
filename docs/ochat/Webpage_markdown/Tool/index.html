Module Webpage_markdown.Tool

Convert a web page to Markdown and expose the operation as a GPT-tool.

  • Overview
  • Behaviour
  • Example

Overview

This module couples the declarative specification Definitions.Webpage_to_markdown with a concrete implementation that fetches the page, converts it to Markdown, and caches the result. The single entry-point register produces a Ochat_function.t value ready to be passed to Ochat_function.functions (and ultimately to Openai.Completions.post_chat_completion).

val register : env:Eio_unix.Stdenv.base -> dir:_ Eio.Path.t -> net:_ Eio.Net.t -> Ochat_function.t

register ~env ~dir ~net returns the concrete implementation of the "webpage_to_markdown" GPT-tool.

The function wraps the real work in a small in-memory cache. Re-invoking the tool on the same URL within five minutes returns the cached Markdown rather than re-downloading the page.
