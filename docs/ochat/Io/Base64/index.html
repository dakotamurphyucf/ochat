Module Io.Base64

Encode a local file into a "data:" URI.

val file_to_data_uri : dir:Eio.Fs.dir_ty Eio.Path.t -> string -> string

file_to_data_uri ~dir file loads file from dir, Base-64 encodes its contents and prefixes the result with the proper MIME type deduced from the extension.
