Module Preprocessor

Prompt Preprocessor ------------------ This module provides a transparent hook that can be inserted right before the ChatMarkdown parsing step. When enabled, the hook runs the Recursive Meta-Prompting refinement loop on the raw prompt string. Enablement happens via either:

1. Environment variable OCHAT_META_REFINE set to a truthy value ("1", "true", "yes", "on"). 2. Presence of the sentinel comment "<!-- META_REFINE -->" within the prompt.

The transformed prompt preserves ChatMarkdown validityâ€”the added metadata is encoded as HTML comments so that downstream parsers are oblivious to it.
