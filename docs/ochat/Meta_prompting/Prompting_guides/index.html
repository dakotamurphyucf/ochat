Module Meta_prompting.Prompting_guides

Prompting guides and best-practice references.

This module contains a set of large, multi-line strings that act as reference guides when constructing prompts for Large Language Models (LLMs). They are intended to be embedded verbatim into the system or developer messages that steer an agent.
