val default : unit -> t

default () returns a fresh context with

  • rng seeded deterministically for reproducible test runs;
  • model_to_optimize defaulting to Model.model.O3;
  • action = Generate;
  • prompt_type = General.

Use this when migrating existing code that did not yet thread an explicit context value.

val with_proposer_model : t -> model:Model.model option -> t

with_proposer_model t ~model returns a shallow copy of t in which proposer_model has been replaced by model. All other fields are kept intact.

val with_guidelines : t -> guidelines:string option -> t

with_guidelines t ~guidelines returns t with the guidelines field replaced by guidelines.
