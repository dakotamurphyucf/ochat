val is_expired : t -> bool

is_expired t returns true if t will expire within the next minute.

A small *safety window* (60&nbsp;s) is subtracted from expires_in so that callers can refresh the token a bit early and avoid race conditions where a request is dispatched with a token that expires en-route.
