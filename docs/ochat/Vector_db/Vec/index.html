val write_vectors_to_disk : t array -> path -> unit

write_vectors_to_disk vecs path writes vecs to path using Bin_prot. The file is created (or truncated) with mode `0o600`.

The function runs in the calling fibre and therefore must not be used inside a preemptively blocking context.

val read_vectors_from_disk : path -> t array

read_vectors_from_disk path deserialises an array previously written by write_vectors_to_disk.
