Module Chat_tui.Ui_helpers

Terminal-side helpers that are **outside** the Notty event loop. They are intended to be called **after** the TUI has shut down, e.g. from Chat_tui.App.run_chat's teardown logic. Currently the module exposes a single helper ‚Äì prompt_archive.
