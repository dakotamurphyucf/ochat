Module App_reducer.Context

run ... runs the main event loop and blocks until the UI should stop.

The reducer is single-threaded with respect to the UI model: it is the only place that should mutate runtime.model. Background fibres communicate by pushing App_events.internal_event values into the internal event stream.

All inputs are bundled in Chat_tui.App_reducer.Context.t.

  • returns

    true iff the user quit via Esc while idle.

Example (as used by Chat_tui.App.run_chat):

  let quit_via_esc = Chat_tui.App_reducer.run ctx in
  ignore quit_via_esc
