Module Chat_tui.App_compaction

Spawn user-triggered history compaction.

The terminal UI supports semantic compaction of long conversations via Context_compaction.Compactor.compact_history. This module starts the compaction worker and reports completion back to the reducer via Chat_tui.App_events.internal_event values.
