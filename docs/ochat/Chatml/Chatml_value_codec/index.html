Module Chatml.Chatml_value_codec

Shared codecs between ChatML runtime values and host-facing data representations.

The current focus of this module is conversion to and from Jsonaf.t, plus a few small validation helpers commonly needed by host runtimes.

val jsonaf_to_value : Jsonaf.t -> Chatml_lang.value

Convert a Jsonaf.t tree into the builtin ChatML json runtime representation.

val value_to_jsonaf_result : Chatml_lang.value -> (Jsonaf.t, string) Core.result

Convert a ChatML value representing builtin JSON into Jsonaf.t, returning a descriptive error on shape mismatch.

val value_to_jsonaf_exn : Chatml_lang.value -> Jsonaf.t

Exception-raising wrapper around value_to_jsonaf_result.
