Module Chatml.Chatml_eval

Public evaluator entrypoints for resolved ChatML programs.

The evaluator itself operates on the resolved AST emitted by the resolver. Most embedders use eval_program; host runtimes that interpret Chatml_lang.task values also use apply_value_result to invoke ChatML closures or builtin functions from OCaml.

type eval_result =
  1. | Value of Chatml_lang.value
  2. | TailCall of Chatml_lang.clos * Chatml_lang.value list

Intermediate result produced by the evaluator trampoline.

val finish_eval : Frame_env.env -> eval_result -> Chatml_lang.value

Finish evaluation of a possibly tail-calling result in the given frame environment.

val apply_value_result : Chatml_lang.value -> Chatml_lang.value list -> (Chatml_lang.value, Chatml_lang.runtime_error) Stdlib.result

Apply a runtime value as a function.

This supports both ChatML closures and builtin functions and returns a structured runtime error instead of raising. It is the preferred API for host-side task interpreters.

val apply_value_exn : Chatml_lang.value -> Chatml_lang.value list -> Chatml_lang.value

Exception-raising wrapper around apply_value_result.

val eval_expr : Chatml_lang.env -> Frame_env.env -> Chatml_lang.resolved_expr Chatml_lang.node -> eval_result

Evaluate a resolved expression to either a value or a deferred tail-call.

val eval_program : Chatml_lang.env -> Chatml_lang.resolved_program -> unit

Evaluate an entire resolved program into the supplied top-level environment.
