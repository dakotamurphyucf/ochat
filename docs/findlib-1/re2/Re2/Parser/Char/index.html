Module Parser.Char

val any : Base.char t

any matches newline, unlike "." in many regex syntaxes.

val one_of : Base.char Base.list -> Base.char t

Duplicates in the lists given to one_of and not_one_of are ignored.

val not_one_of : Base.char Base.list -> Base.char t

The following 6 values match the Re2 character classes with the same name.

val lower : Base.char t

val alpha : Base.char t

val digit : Base.char t

val alnum : Base.char t

val space : Base.char t

A character matching Char.is_whitespace
