Module Make2_local.Monad_infix

Same as Infix, except the monad type has two arguments. The second is always just passed through.

val (>>=) : ('a, 'e) X.t -> ('a -> ('b, 'e) X.t) -> ('b, 'e) X.t
val (>>|) : ('a, 'e) X.t -> ('a -> 'b) -> ('b, 'e) X.t
