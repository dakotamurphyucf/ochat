Module Bin_prot_utils_eio

Eio-based alternatives to Chatgpt.Bin_prot_utils. All functions that previously took a file name now take an Eio.Path.t rooted at some directory capability (typically Eio.Stdenv.cwd or Eio.Stdenv.fs).
